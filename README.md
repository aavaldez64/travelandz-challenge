# Travelandz Challenge

Proyecto desarrollado por [aavaldez64](https://github.com/aavaldez64) para el desafío técnico sobre Gestionar Transfers con Hotelbeds API.

Objetivo: Crear una aplicación web que permita a los usuarios buscar y gestionar transfers
utilizando la API de Hotelbeds.
